1. DE MINISTERIO ANGLICANO LIBER PRIMVS.

    _ CHAP. I. Proaemium, in quo Iesuitarum, & Sacerdotum Pontificiorum in lucrandis Proselytis ingenium depingitur. Quod ingens opus duplici ratione moliuntur.
